SAP Navigator
A quick reference for SAP support, administration, and navigation.

As an IT consultant working daily with SAP, I kept running into the same frustrating problem: knowing roughly what I needed to do, but spending too much time hunting down the right transaction code or table name. The documentation is scattered, the built-in search is limited, and asking a colleague breaks their flow too.

So I built SAP Navigator — a fast, offline-first search engine across multiple SAP datasets. Describe a task in plain language, enter a code directly, or browse structured troubleshooting flows. No login, no subscription, no waiting.

Transactions
Describe a task in plain language — get the right transaction with related alternatives and a reason for each match. Searches across 16,000+ SAP transaction codes.
Tables
Look up SAP standard table names by name or keyword — useful before writing ABAP queries or naming custom tables.
Decode
Describe a symptom or error — get the likely category, a starting transaction, and numbered troubleshooting steps.
Flows
Step-by-step process guides organised by module. Star the flows you return to often — they stay pinned at the top.

Everything runs in your browser — no backend, no tracking, no account needed.

Last updated: April 2026

1. No Data Collection

SAP Navigator does not collect, store, or transmit any personal data. There is no backend server, no analytics, no cookies, and no tracking of any kind. The tool is a fully static site hosted on GitHub Pages.

2. What Stays on Your Device

The only data written to your browser is stored in localStorage on your own device. This includes:

  • Your favourited flows (a list of flow IDs you have starred)
